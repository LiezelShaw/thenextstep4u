Why HTML, CSS and Bootstrap

With everything today moving towards the internet, it is always a good idea as a software developer to know how to create webpages and make them look good. Today I'll discuss some reasons why it is a good idea to learn how to write in HTML with css and to use Bootstrap to create responsive web pages.

Everything is centred around the internet nowadays and whether you are planning on doing frontend, backend or full stack development, it is always a good idea to know how to create a website or a web application.

Frontend and full stack developers use HTML and CSS quite extensively, but even as a backend developer, you will find it very beneficial to know and understand HTML.

HTML stands for Hyper Text Markup Language. This is the language that web pages are written in. An html page consists of elements that describe the structure of the page and tells the browser how to display the page. The elements tell the browser that this piece of text is a heading or a paragraph or a hyperlink, for example.

Here is an example of a simple HTML document: 

            <!DOCTYPE html>
<html lang="en">
    <body>
        <h1>Heading</h1>
        <p>This is the first paragraph of the html document.</p>
        <a href="https://www.thenextstep4u.com">This is a link to a website</a>
    </body>
</html>

This is how the above document will be displayed as a web page:

HTML on its own is not enough though, as you can see. It looks quite plain and not like the websites you see on the web at all. It only gives you the content and structure, but no formatting. Plain HTML on its own will make a website look quite dreary and sad.

That is where CSS comes in. CSS stands for Cascading Style Sheets. This is where you add the look and feel to your website. CSS tells your page how it should be displayed. You can change font, colour, size, backgrounds, placing on the screen and many more features.

Letâ€™s go back to our example web page, I have added a css stylesheet. You can add this in the same file with <style> tags in the <head> of your document, or as a separate .css file.

  background-color: darkgrey;
}

  color: blue;
  text-align: center;
}

  font-family: verdana;
  font-size: 20px;
}

  font-size: 10px;
}

To further improve your skill and make your website look professional, you can use Bootstrap. Bootstrap is the most popular front-end, open source toolkit to build responsive, mobile first websites.

Here is our example html code with bootstrap included: 

            <html lang="en">
    <head>
        <link rel="stylesheet" href="https://stackpath.bootstrapcdn.com/bootstrap/4.5.0/css/bootstrap.min.css"
              integrity="sha384-9aIt2nRpC12Uk9gS9baDl411NQApFmC26EwAOH8WgZl5MYYxFfc+NcPb1dKGj7Sk" crossorigin="anonymous">
        <title>Page Title</title>
        <style>
            body {
              background-color: darkgrey;
            }
            h1 {
              color: blue;
              text-align: center;
            }
            p {
              font-family: verdana;
              font-size: 20px;
            }
            .smalltext {
              font-size: 10px;
            }
        </style>
    </head>
    <body>
        <div class="m-4">
            <h1>This is the heading</h1>
        </div>
        <div class="container">
            <div class="row">
                <div class="col-4">
                    <div class="card">
                        <div class="card-body">
                            <h3 class="card-title">Column 1</h3>
                            <p class="card-text">Now you can divide your page into columns.</p>
                            <a href="#" class="btn btn-primary">Go somewhere</a>
                        </div>
                    </div>
                </div>
                <div class="col-4">
                    <div class="card">
                        <div class="card-body">
                            <h3 class="card-title">Column 2</h3>
                            <p class="card-text">Now you can divide your page into columns.</p>
                            <a href="#" class="btn btn-primary">Go somewhere</a>
                        </div>
                    </div>
                </div>
                <div class="col-4">
                    <div class="card">
                        <div class="card-body">
                            <h3 class="card-title">Column 3</h3>
                            <p class="card-text">Now you can divide your page into columns.</p>
                            <a href="#" class="btn btn-primary">Go somewhere</a>
                        </div>
                    </div>
                </div>
            </div>
        </div>
    </body>
</html>

It might look complicated, but it is not really complicated. I've just added a few Bootstrap class tags that they give you out the box to style the document.
